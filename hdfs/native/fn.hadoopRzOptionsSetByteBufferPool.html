pub unsafe extern fn hadoopRzOptionsSetByteBufferPool(opts: *const hadoopRzOptions, className: *const c_char) -> c_int

Set the ByteBufferPool to use with read0.

Params

  • opts - The options structure.
  • className - If this is NULL, we will not use any ByteBufferPool. If this is non-NULL, it will be treated as the name of the pool class to use. For example, you can use ELASTIC_BYTE_BUFFER_POOL_CLASS.

Return

0 if the ByteBufferPool class was found and instantiated; -1 plus errno otherwise.
