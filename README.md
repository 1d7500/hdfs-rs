# hdfs-rs


libhdfs binding library and rust APIs which safely wraps libhdfs binding APIs

## Documentation
* [API documentation] (http://hyunsik.github.io/hdfs-rs/)

## Usage
Add this to your Cargo.toml:

```toml
[dependencies]
hdfs = "0.0.1"
```

and this to your crate root:
```rust
extern crate hdfs;
```
